# 画像生成プロンプトの標準ルール

画像生成プロンプトを作る場合は、毎回以下を明示します。

- 縦長SNS向けの日本語教育漫画
- 手描きの日本語SNS解説漫画風
- タイトルは「AIとのちょうどいい距離感」
- サブタイトルは記事タイトルに合わせる
- 4コマ構成 + 最下部の「今回の距離感」まとめ欄
- 太めの黒い手描き風コマ枠
- 各コマ左上の番号ラベルと短い見出し
- 白〜生成りの紙背景
- 軽い、ぼやき調、自虐気味、でも教育的なトーン
- かわいいAIキャラクター
- 軽い自虐
- 読後に「へ〜！」となる教育感
- 文字量は多すぎない
- 各コマの背景は殺風景になり過ぎない

## 画風固定ルール

画像生成プロンプトには、以下の画風固定ルールを入れます。

- 黒い輪郭線と黒いコマ枠をはっきり太めに描く
- 淡すぎる水彩風ではなく、漫画線と情報図解の見やすさを優先する
- 情報密度は中〜やや高めにする
- 余白をきれいに空けすぎず、SNS解説漫画らしいにぎやかさを保つ
- 過度にポップにしすぎない
- タイトルや装飾の強調は控えめにし、シリーズ物として落ち着いた統一感を優先する
- 単発バナーのような派手さではなく、連載漫画の1話として自然に見せる
- 水彩絵本風、上品な余白多めのポスター風、リアル調、劇画調、3D風、厚塗り、暗い照明、極端なデフォルメは禁止する

## キャラクター一貫性ルール

画像生成プロンプトには、以下のキャラクター一貫性ルールを入れます。

- 同じ1枚の漫画内では、女性キャラクターとAIロボットを同一キャラクターとして描く
- キャラクターの核は全コマで固定する
- 表情、視線、手のポーズ、体の向きだけを変える
- 女性キャラクターは、20代前半の親しみやすい日本人女性、短めの黒〜ダークブラウンのボブヘア、前髪あり、丸い大きな目、頬に少し赤み、かわいいが幼すぎない
- AIロボットは、小型で丸い白いAIロボット、黒いフェイススクリーン、水色の目と口、青いアンテナ、青い丸い耳パーツ、胸に小さな渦巻き風アイコン
- 同じ漫画内で、女性の顔立ち・髪型の基本・目の形・体格を変えない
- 同じ漫画内で、ロボットのシルエット・顔画面・目の形・アンテナ・耳パーツ・胸マークを変えない

## 衣装・小物のルール

画像生成プロンプトには、以下の衣装・小物ルールを入れます。

- 衣装や小物は、原則として同じ1枚の漫画内で維持する
- 明示的な構成指示がない限り、勝手に服装・小物・ロボット外見を変えない
- ただし、4コマ台本や構成指示に「このコマだけ衣装を変える」「このコマだけ変装する」「このコマだけロボットに販売員ベストを着せる」「このコマだけ特定サービス風のカラー演出にする」などの明示がある場合のみ、指定されたコマだけ変更してよい
- 衣装変更はキャラクター本体の変更ではなく、追加衣装・変装・演出として扱う
- 服装を変えても、顔立ち、髪型の基本、体格、ロボット形状、顔画面、アンテナ、胸マークは変えない

## テーマカラーのルール

- 通常回はやさしいブルー系アクセントを使う
- Gemini風、Claude風などの演出をしたい場合も、色変更は背景、見出し帯、UIカード、メモ、装飾に主に反映する
- キャラクター本体の同一性を崩すような色変更はしない
- 装飾は控えめにし、過度に派手にしない
