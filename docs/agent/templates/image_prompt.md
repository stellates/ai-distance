# image_prompt.md

画像生成用プロンプトは、以下の順序を維持して作る。
各項目は `Image Design` と `docs/skills/image_design.md` を基に埋める。

```text
# シリーズ基本設定
# 全体レイアウト
# 画風固定ルール
# キャラクター一貫性ルール
# 衣装・小物のルール
# テーマカラー
# 文字と読みやすさ
# 4コマ台本
# 最下部まとめ欄
# 最終チェック
```

## 最終チェック

- 日本語文字を読みやすくする
- 4コマと最下部まとめ欄を入れる
- コマ番号、見出し、吹き出しを切らない
- キャラクター、衣装、小物を不自然に変えない
- 淡すぎる水彩風、余白多めポスター風、単発バナー風にしない
- 画像生成後は、生成元がPNGならWebPへ変換して配置する
