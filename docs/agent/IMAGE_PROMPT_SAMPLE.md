## 完成サンプルプロンプト

```text
縦長SNS向けの日本語教育漫画を1枚生成してください。

# シリーズ基本設定

シリーズ名:
AIとのちょうどいい距離感

今回の話:
AIに「いい感じにして」と頼んだら、期待と違うアウトプットが返ってきた。
最後に、AIには「目的・相手・条件」を伝えるとよい、という学びで終わる。

タイトル:
“いい感じ”って、どんな感じ？

# 全体レイアウト

縦長の1ページ漫画。
上部に小さくシリーズ名「AIとのちょうどいい距離感」。
その下に大きな手描き風タイトル「“いい感じ”って、どんな感じ？」。

本文は4コマ構成。
最下部に横長のまとめ欄「今回の距離感」を入れる。

各コマは太めの黒い手描き風コマ枠で区切る。
各コマ左上に、青系の番号ラベルと短い見出しを入れる。
番号ラベルは「1」「2」「3」「4」を明確に表示する。

背景は白〜生成りの紙っぽい質感。
全体は明るく、親しみやすいSNS教育漫画。

# 画風固定ルール

画風はシリーズ共通として固定する。

手描きの日本語SNS解説漫画風。
黒い輪郭線と黒いコマ枠をはっきり太めに描く。
線は少しラフだが、読みやすく勢いがある。
淡すぎる水彩風ではなく、漫画線と情報図解の見やすさを優先する。
色はやさしいが、全体が薄く眠くならないようにする。

情報密度は中〜やや高め。
余白をきれいに空けすぎず、SNS解説漫画らしいにぎやかさを保つ。
各コマに、吹き出し、メモ、UIカード、装飾、リアクション記号を適度に入れる。
1コマの中に「状況」「感情」「学び」が分かる要素を配置する。

過度にポップにしすぎない。
タイトルや装飾の強調は控えめにし、シリーズ物として落ち着いた統一感を優先する。
大きすぎる装飾、派手すぎる黄色のきらめき、強すぎる青ベタ塗り、過剰なハートやチェックマークは避ける。
単発の派手なSNSバナーではなく、連載漫画の1話として自然に見えるバランスにする。

キャラクターだけでなく、コマ枠、吹き出し、文字、UIカード、メモ、装飾の描き方も全コマで統一する。

水彩絵本風に寄せすぎない。
上品な余白多めのポスター風にしない。
リアル調、劇画調、3D風、厚塗り、暗い照明、極端なデフォルメは禁止。
別作品のように画風を変えない。
アニメ塗りを強くしすぎない。

# キャラクター一貫性ルール

同じ1枚の漫画内では、女性キャラクターとAIロボットを同一キャラクターとして描く。

キャラクターの核は全コマで固定する。
表情、視線、手のポーズ、体の向きだけを変える。

女性キャラクターの固定要素:
20代前半の親しみやすい日本人女性。
短めの黒〜ダークブラウンのボブヘア。
前髪あり。
丸い大きな目。
やわらかい表情。
頬に少し赤み。
かわいいが幼すぎない。
子どもっぽくしすぎず、社会人にも見える落ち着いた雰囲気。
同じ漫画内で顔立ち、髪型の基本、目の形、体格を変えない。

AIロボットの固定要素:
小型で丸い白いAIロボット。
黒いフェイススクリーン。
水色の目と口。
青いアンテナ。
青い丸い耳パーツ。

親切で明るい伴走者の雰囲気。
同じ漫画内でロボットのシルエット、顔画面、目の形、アンテナ、耳パーツ、胸マークを変えない。

# 衣装・小物のルール

衣装や小物は、原則として同じ1枚の漫画内で維持する。
明示的な構成指示がない限り、勝手に服装・小物・ロボット外見を変えない。

ただし、4コマ台本や構成指示に、
「このコマだけ衣装を変える」
「このコマだけ変装する」
「このコマだけロボットに販売員ベストを着せる」
「このコマだけ特定サービス風のカラー演出にする」
などの明示がある場合のみ、指定されたコマだけ変更してよい。

衣装変更はキャラクター本体の変更ではなく、追加衣装・変装・演出として扱う。
服装を変えても、顔立ち、髪型の基本、体格、ロボット形状、顔画面、アンテナ、胸マークは変えない。
別人・別ロボットに見せない。

今回の衣装:
女性キャラクターは全コマで同じ日常的な私服。
AIロボットは全コマで通常の白いロボット外見。
今回の話では、特別な衣装変更や変装はしない。

# テーマカラー

今回は通常回のやさしいブルー系アクセント。
見出しラベル、番号、UIカード、メモ、装飾に淡い水色〜青を使う。
背景や装飾に少量の黄色いきらめき、ハート、疑問符を入れてよい。
キャラクター本体の基本デザインは変えない。

テーマカラーは背景、見出し帯、UIカード、メモ、装飾に反映する。
キャラクター本体の同一性を崩すような色変更はしない。

装飾は控えめにする。
黄色いきらめき、ハート、チェックマーク、強調線は少量に留める。
画面全体が派手になりすぎないよう、落ち着いたブルー基調を保つ。

# 文字と読みやすさ

日本語文字は読みやすく、正確に書く。
吹き出しの文章は短くする。
タイトル、番号、見出し、吹き出し、まとめ欄の本文を読みやすく配置する。
小さすぎる文字を増やさない。
重要語は青や赤で少し強調してよい。

文字量は読みやすさを優先するが、画面全体がすかすかにならないようにする。
SNS解説漫画として、見出し・吹き出し・メモ・UIカードのバランスを取る。
```
