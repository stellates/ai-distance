# POST_INVENTORY.md

このファイルは、「AIとのちょうどいい距離感」の既存記事を棚卸しし、新記事作成時の重複チェックを省力化するための一覧です。

本文の代替ではありません。
テーマが近い記事が見つかった場合は、該当する `posts/NNN.html` を確認してから新テーマを判断してください。

---

## 運用ルール

- 1記事1行で管理する
- 最新記事を上に追加する
- 本文を長く引用しない
- 重複チェックに必要な要約だけを書く
- 新記事作成時は、テーマ提案前に必ずこのファイルを読む
- 新記事追加時は、ChatGPT が追記行を出力し、人間がローカル clone 上で手動反映する
- `Theme` と `Distance` は、似た教訓の重複を見つけやすいように短く具体的に書く
- `Similar / Avoid` には、似ている既存記事や重複回避メモを書く

---

## 追記方法

新記事追加時、ChatGPT は以下の形式で1行を出力します。

```md
| NNN | YYYY-MM-DD | タイトル | 何のAIあるあるか | 今回の距離感・教訓 | タグ1, タグ2, タグ3 | 似ている既存記事・重複回避メモ |
```

人間は、出力された1行を `## 記事一覧` のテーブル先頭へ貼り付けます。
